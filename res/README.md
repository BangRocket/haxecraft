# Resource Layout

Runtime assets for the MMO client and zone server, under `res/`.

## Contents

- `res/sprites/` — sprite sheets loaded at runtime (`.png`).
- `res/maps/` — Tiled `.tmx` maps.

## Sprite Sheets

The client renders from these category sheets:

- `sprites/sprites_terrain.png`
- `sprites/sprites_player.png`
- `sprites/sprites_items.png`

## Maps

- `maps/starter.tmx` — the procgen starter zone, regenerated by
  `make regenerate-map` (see `tools/worldgen-tmx`).

## Loading

Sheets are decoded from disk with `sys.io.File` + `hxd.res.Any.fromBytes`
(see `client.render.ZoneRenderer`); the zone parses the `.tmx` via
`shared.world.TmxParser`.
